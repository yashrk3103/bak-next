# Required images

Place the following images in this folder:

| File | Used in | Description |
|------|---------|-------------|
| `hero-bg.jpg` | HeroSection | Full-width hero background (qurbani image) |
| `logo.png` | Navbar | Zirwa Qurbani Service logo (transparent PNG) |

These are referenced in the code as `/images/hero-bg.jpg` etc.
Until the images are added, the hero shows a dark red fallback background.
